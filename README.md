## 🛠️ Using the Mod Manager

The interface displays your mods in a single list. You can add new mods by dragging them from your desktop into the Mod Manager window. 

The software checks for conflicts between your mods. If two mods change the same game file, the manager highlights them in red. This helps you prevent game crashes by keeping your files compatible.

Use the checkbox next to each mod to enable or disable it. The order of the list determines which mod the game loads first. You can click and drag rows to change this order. Place critical files at the top of the list.

## ❓ Frequently Asked Questions

What should I do if a mod causes a crash?
Disable the most recent mods one by one. Restart the game after each change to see if the error persists.

Do I need to update the mods?
Yes. When the game developers release an update, mod makers often release new versions. Check this tool periodically for updates to your specific mods.

Is this tool safe for my game save?
Always create a backup of your save folder before you install new mods. While the manager works to keep things stable, file corruption can happen with complex mods.

Does this work with online modes?
No. Modding EA Sports FC 26 poses a risk to online play. Use this tool for offline modes like Career and Tournament. Disable all mods before you access online features to prevent account bans.

## 💡 Troubleshooting

If the application does not start, check that you have the latest .NET Runtime installed on your system. You can find this on the official Microsoft website. If the manager fails to find your game, manually select the game folder by clicking the Settings button within the application menu.
